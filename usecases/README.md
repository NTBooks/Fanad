# Fanad use cases

Real ways people use Fanad, one article each. They're independent — start with whichever matches your life — but the first one explains the core loop everything else builds on.

| Article | What it covers |
|---|---|
| [Getting it out of your head](getting-it-out-of-your-head.md) | The core loop: brain-dump everything, ask for one thing to do now, let stale tasks sleep. Read this one first. |
| [Diet and meal tracking](diet-and-meal-tracking.md) | Calorie logging that converges to three words a meal: confirm-once foods, saved meals, recipes by cooked weight. |
| [The problem-solving journal](problem-solving-journal.md) | Finding slow patterns — "is it the dairy?" — with a daily checklist, a one-line note, and AI trend summaries that name their evidence. |
| [Nested lists and groceries](nested-lists-and-groceries.md) | The outliner for things that aren't tasks: groceries by aisle, packing lists, anything with an inside. |
| [Planning a party with steps](planning-a-party-with-steps.md) | Breaking a big vague task into a checklist, letting Fanad draft the steps with `guess`, and saving the shape as a template. |
| [Household memory](household-memory.md) | Notes you get back by meaning, not exact words: the spare key, the wifi password, the furnace filter photo. |
| [Reminders and deadlines](reminders-and-deadlines.md) | Putting time on tasks in plain words — `by`, `on`, `remind me` — plus today-scoping, .ics calendar handoff, and the at-most-one-ping promise. |
| [Kitchen timers](kitchen-timers.md) | The one-shot ding that never touches your list: pasta, laundry, and the 20-minute "just start" trick. |
| [Routines without recurrence](routines-without-recurrence.md) | Why Fanad has no recurring tasks, and what to use instead: templates, your own calendar via .ics, one-shot reminders, daily check-ins. |
| [Notebooks: separate spaces](notebooks-separate-spaces.md) | Isolated sub-spaces for a big project, a work/home split, or someone you're helping. |

All commands shown work with or without the leading slash. Modules (Diet, Journal, Lists, Notes, Notebooks, Timer) are per-person and off by default — each article starts with its `optin`.
