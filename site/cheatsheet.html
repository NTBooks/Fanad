Fanad

Get it out of your head.
The one rule: make a statement → it’s filed as a task. Ask a question → it’s answered. The / prefix is always optional — it just escapes an open question. Not a chatbot: one thought per line — a whole paragraph files as a single task.

🌐 The Web App — for more control

/webtype it in Telegram or Slack → a one-time sign-in link that opens Fanad in your browser, already signed in as you. It works once and expires in minutes — run /web again anytime.
cmdtype it in any chat (web or a Telegram/Slack DM) → a ready-to-paste npx github:NTBooks/Fanad <server> <token> command for the terminal client (a full-screen chat in your console). It runs in any terminal with Node.js 24+; npx fetches the client from the repo, so no checkout is needed. Admin opt-in: the owner enables it under Settings → Security → Terminal client tokens. The token is shown once; choose how long it stays valid (30 / 90 days, 1 year, or never) when you mint it, and revoke anytime in the same place.
tokentype it in any chat → a warning + yes/no, then a read-only, never-expiring token scoped to your own account (shown once). It's the credential a Home Assistant dashboard needs: it can only read /api/ha/summary, never post or change anything. Any authorized user can self-mint; same enable switch as cmd. Copy it, then delete the message.
Everything on this sheet works in the web chat too, plus GUI views the phone can’t do: kanban board for tasks, notes wall, metric charts, lists outliner, templates, diet report with inline food-log edits, the 📔 journal, and 🧪 batches — toggle GUI/Text per view, pick a theme, and manage your modules in Settings.
On a wide screen the web chat also floats side panels in the empty space beside the column: a shortcut legend on the left (tap a letter to start that command) and a status panel on the right — your module toggles, the task you’re working on, upcoming timers & reminders, and today’s notebook / mood / date. Both follow your module opt-ins, and one » button hides the whole layer at once.
Fanad — your assignment pad. Local-first; your data stays on your box. Tasks are always on; every other module is off until you optin. Full manual: h in chat, or 📖 in the web app.
